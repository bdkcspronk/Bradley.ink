# Bradley Ink

Basic static website scaffold ready to deploy on Netlify.

## Pages

- `index.html` — homepage and contact.
- `portfolio.html` — portfolio placeholders for future project case studies.

## Local preview

```bash
python3 -m http.server 8080
```

Then open:

- <http://localhost:8080>
- <http://localhost:8080/portfolio.html>
